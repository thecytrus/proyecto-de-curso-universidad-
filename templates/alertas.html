Gestión de Alertas

Ayuda para crear nuevas alertas
- El umbral es el valor limite que debe alcanzar la medida para que se dispare la alerta. Por ejemplo, si el umbral es 30 y el usuario ingresa una condicion "Mayor que(>)", además de seleccionar tipo de medida "Temperatura Ambiente", se monitoreará si alcanza un valor mayor que 30 grados Celsius, en caso de cumplir la condición se disparará la alerta.

- El tipo de alerta es el tipo de medida que se va a monitorear. Por ejemplo, si se selecciona "Humedad del Suelo", se monitoreará solamentela humedad del suelo.
